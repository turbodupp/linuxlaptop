### 1. Issues and workarounds

1.1: Unable to git push, ssh, key error  
  Start the sshkey-agent:  

  `eval "$(ssh-agent -s)"`  
  
  add the ssh key:  

  `ssh-add ~/%pubkey%`  

  
1.2: High fan speed/temp  
Check out [throttled](https://github.com/erpalma/throttled)
